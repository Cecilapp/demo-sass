# Sass demo

> [Sass](https://sass-lang.com) demo for the [Cecil](https://cecil.app) static site generator.

## Download Cecil

```bash
curl -LO https://cecil.app/cecil.phar
```

## Serve locally

```bash
php cecil.phar serve --clear-cache=css
```
